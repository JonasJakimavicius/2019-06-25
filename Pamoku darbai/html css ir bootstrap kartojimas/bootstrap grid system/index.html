Bootstrap grid system

Dokumentacija

Stulpelių aukščiaims sukurkite savo klases ir jas pritaikykite. Fonams naudokite 8 bootstrap fono spalvas

Užduotys su panaudoujant .row, .col ir .col-{1-12} variacijas

  1. Atvaizduokite 4 vienodo ploto stulpelius eilutėje.
  2. Atvaizduokite 3 vienodo ploto stulpelius eilutėje.
  3. Atvaizduokite 2 stulpelius eilutėje santykiu 1:2.
  4. Atvaizduokite 3 stulpelius eilutėje santykiu 2:1:1.
  5. Atvaizduokite 4 stulpelius eilutėje santykiu 3:1:1:1.
  6. Atvaizduokite 4 stulpelius. Paskutiniam stulpeliui išskirkite 2/3 ploto o likusiems išdalinkite plotą po lygiai.
  7. Atvaizduokite 3 eilutes po 2 stulpelius, kurie užimtų po pusę eilutės pločio.
  8. Atvaizduokite 9 lygius stulpelius 3 eilutėse.
  9. Atvaizduokite 9 stulpelius 3 eilutėse, kur pirmasis eilutės stulpelis užimtų 2/3 ploto.

Grid system responsive užduotys: .col-{1-size-12}

  1. Atvaizduokite 2 stulpelius eilutėje. Kurie ant sm ir xs ekranų užims po atskirą eilutę.
  2. Atvaizduokite 3 stulpelius eilutėje. Kurie ant sm ir xs ekranų užims po atskirą eilutę.
  3. Atvaizduokite 3 stulpelius eilutėje. Kur ant sm ir xs ekranų pirmasis užims visą eilutę, o likę pasidalis antrąją pusiau.
  4. Atvaizduokite 3 stulpelius eilutėje. Kur ant sm ir xs ekranų paskutinis stulpelis užims visą eilutę, o likę pasidalis antrąją santykiu 1:2 atitinkamai.
  5. Atvaizduokite 12 stulpelių kurie užims skirtingus eilutės pločius prie ekranų dydžių atitinkamai:
    1. Nuo xs iki sm - stulpelis per visą eilutę
    2. Nuo md - 2 sulpeliai eilutėje
    3. Nuo lg - 3 stulpeliai eilutėje
    4. Nuo xl - 4 stulpeliai eilutėje

Grid system kompleksinės užduotys kuriant eilutes stulpeliuose: .row>.col-{1-12}+col{1-12}>.row.col-{1-12}*x

  1. Atvaizduokite pusės pločio stulpelį-bloką kairėje pusėje, o dešinėje 2 eilutes po pusę stulpelio-bloko aukčio
  2. Atvaizduokite pusės pločio stulpelį-bloką dešinėje pusėje, o kairėje 3 eilutes po trečdalį stulpelio-bloko aukčio
  3. Atvaizduokite pusės pločio stulpelį-bloką kairėje pusėje, o dešinėje 2 eilutes po pusę stulpelio-bloko aukčio, kuriose būtų po 3 stulpelius
  4. Atvaizduokite pusės pločio stulpelį-bloką dešinėje pusėje, o kairėje 3 eilutes po trečdalį stulpelio-bloko aukčio, kuriose būtų po 2 stulpelius

Grid system kompleksinės užduotys

Iki MD
Nuo MD
Nuo XL
