Padding - kamšalas

Naudodami TIK VIENĄ css savybę nustatykite:
  1. Kamšalą elementams visose pusėse 10px
  2. Kamšalą elementams šonuose 10px
  3. Kamšalą elementams viršuje ir apačioje 10px
  4. Kamšalą elementams visur tik ne dešinėje 10px
  5. Kamšalą elementams visur tik ne apačioje 10px
  6. Kamšalą elementams šonuose 20px, viršuje 10px ir apačioje 5px

Margin - paraštės

  1. Atvaizduokite 5 raudonus blokelius eilutėje atskirtus 10px tarpais
  2. Atvaizduokite 3 mėlynus blokelius stulpeliu atskirtus 20px tarpais
  3. Atvaizduokite 200x200 žalią bloką viduryje puslapio
  4. Atvaizduokite 200x200 raudoną bloką dišinėje puslapio

Kompleksinės užduotys

1. Atvaizduokite 5 žalius blokelius eilutėje atskirtus 10px tarpais ir išlygiuotus dešinėje ekrano pusėje
2. Padarykite tokį vaizdą viduryje ekrano
