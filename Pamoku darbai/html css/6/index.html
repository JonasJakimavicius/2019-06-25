Display: inline || Elementų atvaizdavimas raidžių rašymo tvarka

Elementai kurių stiliaus savybės display reikšmė yra inline rikiuojami vienas paskui kitą. Iš kairės į dešinę, o jeigu netelpa perkeliami į kitą eilutę. display: inline elementai negali turėti didesnio ar mažesnio aukščio negu jų turinys, nes jie yra traktuojami kaip simboliai. Taip pat jiems negalime nustayti paraščių viršuje bei apačioje. (margin). Iš esmės display: inline elementus naudojame tektiniam turinui.

Elementai kurių stiliaus savybės display reikšmė yra inline-block rikiuojami vienas paskui kitą. Iš kairės į dešinę, o jeigu netelpa perkeliami į kitą eilutę. Šio atvaizdavimo elementai gali turėti apibrėžtą aukšti, plotį, paraštes ir kamšalą (margin & padding). Į kitą eilutę jie persikelia kaip nejudinami vienetai, atvirkščiai negu display: inline elementai.

Pozicionavimas

Static

Tai toks pozicionavimo būdas, kuomet elementai pozicionuojami iš kairės į dešinę, o jeigu netelpa eilutėje yra nukeliami į sekančią eilutę

Relative

Tai pozicionavimas, kuomet top, left, bottom, right css savybės nurodo atsistūmimus nuo elemento static pozicijos.

left: 50px


Absolute

Šį pozicionavimo tipą turintys elementai ieško pirmo, medžio hierarchijoje, tėvo linijoje sutikto NE static elemento ir atsispiria nuo jo per nurodytus dydžius top, right, bottom, left.

Fixed

Atsispiria per nurodytus dydžius nuo viewport'o

Sticky
